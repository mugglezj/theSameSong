# theSameSong
多人在线听歌
可以随时切歌
跟朋友分享歌曲的正确姿势

歌源来自 QQ 音乐


### 目录
```
├─bin            // 后端
│    app.js      // 后端 服务器
│    socket      // 后端 socket
|    router      // 后端 路由
|    libs        // 工具库
|    crawl       // 爬虫
|    controller  // 后端 控制器
|    config      // 后端  配置文件
├─build          // 前端 编译之后 线上目录
├─public         // 前端模板
├─src            // 前端源码
│    Component   // 前端组件
│    App.js      // 前端主路由


├─static       // 旧版前端代码
├─view         // 前端 页面
├─app.js       // 前端 服务器
├─package.json
```

### run

```
cd theSameSong //打开目录
npm i // 安装依赖
npm run build // 编译前端代码
npm run start // 开始运行
```
